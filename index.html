Tailwind CSS Basics

Essential Tailwind utility classes for everyday development. A quick reference for spacing, sizing, colors, typography, and more.

Spacing

Padding and margin utilities. The scale uses 0.25rem (4px) increments.

Padding
p-4    /* 1rem all sides */
px-4   /* 1rem left + right */
py-4   /* 1rem top + bottom */
pt-4   /* 1rem top */
pr-4   /* 1rem right */
pb-4   /* 1rem bottom */
pl-4   /* 1rem left */
Margin
m-4    /* 1rem all sides */
mx-4   /* 1rem left + right */
my-4   /* 1rem top + bottom */
mt-4   /* 1rem top */
mr-4   /* 1rem right */
mb-4   /* 1rem bottom */
ml-4   /* 1rem left */
Common Scale Values
p-0    /* 0px */
p-1    /* 0.25rem (4px) */

/* Negative values */

/* Auto margins */
mx-auto    /* center horizontally */
ml-auto    /* push to right */

Responsive Prefixes

Apply styles at specific breakpoints. Tailwind is mobile-first: unprefixed utilities apply to all sizes, prefixed utilities apply at that breakpoint and above.

Breakpoints
sm:   /* 640px and up */
md:   /* 768px and up */
lg:   /* 1024px and up */
xl:   /* 1280px and up */
2xl:  /* 1536px and up */
Usage Examples
/* Stack on mobile, row on larger screens */
flex flex-col md:flex-row

/* Hide on mobile, show on desktop */
hidden lg:block

/* Responsive padding */
p-4 md:p-8 lg:p-12

Mobile-First Principle

Start with mobile styles (no prefix), then add larger breakpoint overrides. text-sm md:text-base lg:text-lg means: small text on mobile, base on tablets, large on desktop.

CSS Flexbox → Tailwind

A quick reference guide mapping CSS Flexbox properties to their Tailwind CSS utility classes.

Parent Properties Child Properties

Parent (Flex Container) Properties

display

container

Defines a flex container, enabling flex context for all direct children.

CSS
.container {
  display: flex;
}

.container {
  display: inline-flex;
}
Tailwind
flex          /* display: flex */
inline-flex   /* display: inline-flex */

flex-direction

container

Establishes the main axis, defining the direction flex items are placed in the container.

CSS
.container {
  flex-direction: row;          /* default */
  flex-direction: row-reverse;
  flex-direction: column;
  flex-direction: column-reverse;
}
Tailwind
flex-row          /* left → right */
flex-row-reverse  /* right → left */
flex-col          /* top → bottom */
flex-col-reverse  /* bottom → top */

flex-wrap

container

Controls whether flex items are forced onto one line or can wrap onto multiple lines.

CSS
.container {
  flex-wrap: nowrap;       /* default */
  flex-wrap: wrap;
  flex-wrap: wrap-reverse;
}
Tailwind
flex-nowrap       /* single line */
flex-wrap         /* wrap top → bottom */
flex-wrap-reverse /* wrap bottom → top */

flex-flow

shorthand

Shorthand for flex-direction + flex-wrap. In Tailwind, just combine both classes.

CSS
.container {
  flex-flow: row nowrap;    /* default */
  flex-flow: column wrap;
}
Tailwind
flex-row flex-nowrap  /* combine classes */
flex-col flex-wrap    /* combine classes */

justify-content

container

Defines alignment along the main axis. Distributes extra space when items don't use all available space.

CSS
.container {
  justify-content: flex-start;   /* default */
  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between;
  justify-content: space-around;
  justify-content: space-evenly;
}
Tailwind
justify-start    /* pack at start */
justify-end      /* pack at end */
justify-center   /* center items */
justify-between  /* first/last at edges */
justify-around   /* equal space around */
justify-evenly   /* equal space between */

align-items

container

Defines alignment along the cross axis (perpendicular to main axis). Think of it as justify-content for the cross axis.

CSS
.container {
  align-items: stretch;     /* default */
  align-items: flex-start;
  align-items: flex-end;
  align-items: center;
  align-items: baseline;
}
Tailwind
items-stretch   /* fill container */
items-start     /* align to start */
items-end       /* align to end */
items-center    /* center on cross axis */
items-baseline  /* align baselines */

align-content

container

Aligns flex lines when there's extra space on the cross axis. Only applies when flex-wrap creates multiple lines.

CSS
.container {
  align-content: flex-start;
  align-content: flex-end;
  align-content: center;
  align-content: stretch;      /* default */
  align-content: space-between;
  align-content: space-around;
  align-content: space-evenly;
}
Tailwind
content-start    /* pack lines at start */
content-end      /* pack lines at end */
content-center   /* center lines */
content-stretch  /* stretch to fill */
content-between  /* first/last at edges */
content-around   /* equal space around */
content-evenly   /* equal space between */

gap

container

Controls spacing between flex items. Applies only between items, not at outer edges.

CSS
.container {
  gap: 1rem;              /* both axes */
  gap: 1rem 2rem;         /* row col */
  row-gap: 1rem;
  column-gap: 2rem;
}
Tailwind
gap-4     /* 1rem on both axes */
gap-x-4   /* column gap only */
gap-y-4   /* row gap only */

/* Common values: gap-0, gap-1, gap-2, gap-4, gap-6, gap-8 */

Child (Flex Items) Properties

order

item

Controls the order in which flex items appear. Items with lower order values appear first.

CSS
.item {
  order: 0;    /* default */
  order: 1;
  order: -1;
  order: 99;
}
Tailwind
order-none   /* order: 0 */
order-first  /* order: -9999 */
order-last   /* order: 9999 */
order-1      /* order: 1 */
order-2      /* ... through order-12 */

flex-grow

item

Defines the ability for a flex item to grow if necessary. Accepts a unitless value as a proportion.

CSS
.item {
  flex-grow: 0;    /* default - don't grow */
  flex-grow: 1;    /* grow to fill */
}
Tailwind
grow-0   /* flex-grow: 0 */
grow     /* flex-grow: 1 */

flex-shrink

item

Defines the ability for a flex item to shrink if necessary. Prevents shrinking when set to 0.

CSS
.item {
  flex-shrink: 1;    /* default - can shrink */
  flex-shrink: 0;    /* don't shrink */
}
Tailwind
shrink     /* flex-shrink: 1 */
shrink-0   /* flex-shrink: 0 */

flex-basis

item

Defines the default size of an element before remaining space is distributed. Can be a length or keyword.

CSS
.item {
  flex-basis: auto;     /* default */
  flex-basis: 0;
  flex-basis: 50%;
  flex-basis: 200px;
}
Tailwind
basis-auto   /* auto sizing */
basis-0      /* 0px */
basis-full   /* 100% */
basis-1/2    /* 50% */
basis-1/3    /* 33.333% */
basis-1/4    /* 25% */
basis-64     /* 16rem (256px) */

flex

shorthand

Shorthand for flex-grow, flex-shrink, and flex-basis. Tailwind provides common presets.

CSS
.item {
  flex: 0 1 auto;     /* default (initial) */
  flex: 1 1 0%;       /* grow + shrink */
  flex: 1 1 auto;     /* auto sizing */
  flex: none;         /* 0 0 auto */
}
Tailwind
flex-initial  /* 0 1 auto - shrink only */
flex-1        /* 1 1 0% - grow + shrink, ignore size */
flex-auto     /* 1 1 auto - grow + shrink, consider size */
flex-none     /* 0 0 auto - don't grow or shrink */

💡 Common Pattern

Use flex-1 to make items grow equally regardless of their content size. Use flex-auto when items should grow but respect their intrinsic sizes.

align-self

item

Allows overriding the container's align-items for individual flex items.

CSS
.item {
  align-self: auto;        /* default */
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;
  align-self: baseline;
}
Tailwind
self-auto      /* inherit from container */
self-start     /* align to start */
self-end       /* align to end */
self-center    /* center on cross axis */
self-stretch   /* stretch to fill */
self-baseline  /* align to baseline */
